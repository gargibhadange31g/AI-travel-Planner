AI Travel Planner

AI-powered travel planning web application that generates personalized day-by-day travel itineraries based on destination, duration, budget, and travel preferences.

Features
JWT Authentication (Signup/Login)
AI-generated travel itineraries using Groq AI
Destination images from Unsplash API
Place information from Wikipedia API
Google Maps integration
MongoDB database for storing trips
Responsive React frontend
Secure Express.js backend
